## Packages
framer-motion | Page transitions and layout animations
vaul | High-quality bottom sheet drawer component (standard for mobile)
clsx | Utility for constructing className strings
tailwind-merge | Utility for merging tailwind classes safely

## Notes
- App is designed mobile-first with extensive glassmorphism and depth.
- Using mock data via custom hooks since no backend integration is required.
- All interactive controls enforce a minimum height of 48px for mobile thumb-tapping ease.
- Drawer components use `vaul` for native-feeling mobile bottom sheets.
